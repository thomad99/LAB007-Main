1. Script

YouTube: only works when the video has captions. Whisper: uses OpenAI’s Speech-to-Text API (same OPENAI_API_KEY / OPENAI_KEY as other OpenAI features — billing applies; uploads are not stored on disk). Max file size ~25 MB.

2. Timing & voice

Uses your browser’s speech engine (Web Speech API). Chrome/Edge usually have the best English voices. Click “Refresh voice list” after changing the filter.

3. Build & preview

This text is merged with each segment’s auto keywords before calling the image API (Pexels or fallback), so results stay aligned with your video’s theme.

Subtitles appear here during playback.

Images: set PEXELS_API_KEY on the server for search-matched photos; otherwise a tag-based placeholder service is used. Optional: VIDEOCREATOR_WHISPER_MODEL (default whisper-1). For a downloadable video file, record this tab with OBS or the browser’s screen recorder during Play.

4. Sora (OpenAI video) — optional

Billing and access follow your OpenAI plan. Videos API has strict content policies (e.g. no real-person likeness, no copyrighted characters). Jobs are asynchronous — previews poll about every 12 seconds and can take many minutes for long or Pro renders.
